# Description
Aplikasi Point Of Sales (POS) yang dibuat secara monolith menggunakan framework Laravel dan JQuery

# Fitur yang tersedia pada aplikasi
- Dashboard
- CRUD Kategori
- CRUD Produk
- CRUD Member
- CRUD Supplier
- Modul Pengeluaran
- Modul Pembelian
- Modul Penjualan
- Cetak Nota
- Laporan
